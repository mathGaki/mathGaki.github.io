      2. 산업심리 및 교육

      2.1. 산업심리이론

      2.1.1. 산업심리 개념 및 요소

      ① 산업심리의 개요 ② 심리검사의 종류 ③ 산업안전 심리의 요소

      • 산업안전 심리의 5대 요소 ★★★★★ (동기감습습)
        • 동기(motive) : 능동적인 감각에 의한 자극에서 발생하는 사고의 결과
        • 기질(temper) : 감정적인 경향이나 반응에 관계되는 성격
        • 감정(feeling) : 생활체가 어떤 행동을 할 경우 발생하는 주관적인 동요
        • 습성(habit) : 개체의 대부분에서 관찰할 수 있는 일정한 생활양식
        • 습관(custom) : 성장과정을 통해 형성된 특성
      • 호손(Hawthorne)의 실험 ★
        • 작업 능률은 물리적인 작업조건보다는 인간관계가 더 큰 요소로 작용함

      2.1.2. 인간관계와 활동

      ① 인간관계 ② 인간관계 메커니즘 ③ 집단행동

      • 집단활동의 효과 ★
        • 시너지 효과 : 서로 다른 개체가 힘을 합쳐 개별로 지닌 힘 이상의 결과를 도출하는
        • 동조 효과 : 타인의 행동 또는 주장에 자신의 의견을 일치시키려는
        • 견물 효과 : 타인에게 자신의 장점만 보여주려는
      • 조하리의 창(Johari’s Windows) ★
        • Blind area : 나는 모르지만 다른 사람은 알고 있는 영역
        • Hidden area : 나는 알지만 다른 사람은 모르는 영역
        • Open area : 나는 물론 다른 사람도 알고 있는 영역
        • Unknown area : 나는 물론 다른 사람도 모르는 영역
      • 집단행동
        • 통제적 집단행동 : 관습, 유행, 제도적 행동

      2.1.3. 직업적성과 윤리

      ① 직업적성의 분류 ② 적성검사의 종류 ③ 적성발견 방법 ④ 인사관리

      1. 시각적 판단 검사 : 형태비교, 입체도판단, 언어식별, 평면도판단, 명칭판단, 공구판단검사
      • 적성검사 분석방법
        • 구성타당도
        • 내용타당도
        • 검사-재검사 신뢰도
      • 적성검사를 실시하는 목적
        • 작업자의 생산능률 향상

      2.1.4. 인간행동 성향 및 행동과학

      ① 인간의 일반적인 행동특성 ② 사회행동의 기초 ③ 동기부여 ④ 주의와 부주의

      • 인간의 사회적 행동의 기본형태
        • 대립, 도피, 협력, 융합

      <주의와 부주의>

      • 주의의 특성 ★★★ (변성방)
        • 변동성 : 사람은 한 점에 지속적으로 주의를 집중할 수 없음
        • 선택성 : 사물을 기억하는 3단계(감각보관 > 단기기억 > 장기기억)를 거치면서 입력된 정보를 선택적으로 골라내느 것
        • 방향성 : 정보의 발생방향을 선택한 후 집중적인 정보 입력을 하는 것
      • 부주의의 외적 발생원인
        • 작업 환경조건 불량
        • 작업 순서의 부적합(부자연성)
        • 높은 강도의 직업
        • 기상조건
      1. 강화이론 : 자극, 반응, 보상의 3가지 핵심변인을 가지며, 표출된 행동에 따라 보상을 주는 방식
      2. 역할갈등
        • 역할 마찰 : 역할 내 또는 역할 간의 마찰
        • 역할 부적합 : 집단이 부여한 역할이 개인의 성격이나 능력에 맞지 않을 경우 발생하는 갈등
        • 역할 모호성 : 개인이 수행할 업무 및 책임이 명확하지 않을 경우 발생하는 갈등
        • 역할 과부하 : 역할수행자에 대한 요구가 개인의 능력을 초과하거나 주어진 시간과 능력이 허용하는 것 이상을 달성하도록 요구받고 있다고 느끼는 상황
      3. 래윈(Lewin.K)의 법칙 ★
        • 인간의 행동은 그 사람이 가진 자질 개체와 심리적 환경과의 함수관계
        • B=f(P×E) (\mbox인간의행동=f(\mbox개체×\mbox심리적환경))
        • B(Behavior) : 인간의 행동
        • P(Person) : 개체의 연령, 경험, 심신상태, 성격, 지능 등
        • E(Environment) : 심리적 환경(인간관계, 작업환경(온도, 조명, 소음 등))
      4. 직무스트레스 요인
        • 작업요인 : 작업속도
        • 조직요인 : 관리유형
        • 환경요인 : 소음, 조명
        • 완충작용요인 : 대응능력
      5. 태도
        • 인간의 행동은 태도에 따라 달라지고, 태도가 결정되면 장시간 유지됨
        • 자아방어
        • 자기표현
        • 작업적응
      • 인간의 의식 레벨의 단계별 신뢰성 ★★★
      단계의식의 상태신뢰성의식의 작용
      Phase 0무의식0없음
      Phase 1의식의 둔화0.9 이하부주의
      Phase 2이완 상태0.99 ~ 0.99999마음이 안으로 향함, passive
      Phase 3명료한 상태0.99999 이상전향적, active
      Phase 4과긴장 상태0.9 이하한 점에 집중, 판단정지

      <동기부여>

      • 매슬로우(Abrahaam Maslow) 욕구단계 이론 ★
        1. 1단계 : 생리적 욕구
        2. 2단계 : 안전의 욕구
        3. 3단계 : 사회적 욕구
        4. 4단계 : 존경의 욕구
        5. 5단계 : 자아실현의 욕구
      • 알더퍼(Alderfer)의 ERG 이론
        • 생존욕구(Existence needs) : 다양한 형태의 물리적·생리적 욕구, 인간의 생존을 위한 욕구
        • 성장욕구(Growth needs) : 개인적이며 창조적인 성장을 위한 개인의 노력과 관련된 욕구
        • 관계욕구(Relatedness needs) : 인간답게 살기위해 타인과 관계를 유지하려는 욕구
      • 허츠버그(Herzberg) 2요인(인자) 이론
        • 위생 요인
          • 작업환경에 파생되는 외적이며 물리적인 것(임금, 대인관계, 작업 조건, 보상, 감독)
          • 매슬로우 1-3단계
        • 동기 요인
          • 작업 자체에서 나오는 내적·심리적인 것(성취, 책임감, 인정, 발전)
          • 매슬로우 4-5단계
      1. 데이비스(K.Davis)의 동기부여 이론
        • 지식(Knowledge) x 기능(Skill) = 능력(Ability)
        • 능력(Ability) x 동기유발(Motivation) = 인간의 성과(Human Performance)
        • 인간의 성과(Human Performance) x 물질적 성과 = 경영의 성과
        • 상황(Situation) x 태도(Attitude) = 동기유발(Motivation)
      • 맥그리거(Douglas Macgregor)의 X, Y이론 ★★
        • X이론
          • 사람들은 원래 일하기 싫어하고 일하는 것을 가능한 피하려함
          • 바람직한 목표를 이루기 위해서는 사람들을 통제, 위협, 처벌하는게 필요
          • 사람들은 책임을 회피하고 공식적인 지시, 감독을 선호함
          • 사람들은 명령 받기를 좋아하면 안전을 바라는 인간관을 지님
          • 사람들은 도전적이지 못함
        • X이론의 관리처방
          • 권위주의적 리더십 확립
          • 경제적 보상체제 강화
          • 엄격한 감동 및 통제
          • 상부 책임제도 강화
        • Y이론
          • 사람들은 일하는 것을 자연스럽게 받아들임
          • 외적으로 드러나는 것보다 많은 잠재력을 소유
          • 사람들은 의사결정 능력을 가지고 있으며, 문제 해결 의지를 소유함
          • 사람들은 책임을 수용하고 감수하려는 본성을 지님
          • 조직의 목표에 동의하는 경우 자발적으로 목표 달성을 위해 노력함
        • Y이론 관리처방
          • 분권화와 권한의 위임
          • 목표에 의한 관리
          • 비공식적 조직 활용
          • 자치평가제도의 활성화
      • 안드라고지(Andragogy) 모델
        • 학습자 중심 교육으로 학습자가 자기주도적으로 학습 상황 및 과정을 의미
        • 교슈자는 지원자와 조력자의 역할을 하며, 교수자와 학생이 서로 협력하여 교육계획 및 목표설정, 평가 등이 이루어짐
        • 과업중심, 문제중심, 생활중심의 성향, 학습자의 경험이 가치있는 학습자원으로 여겨짐

      2.2. 인간의 특성과 안전

      2.2.1. 동작특성

      ① 사고경향 ② 안전사고 요인

      1. 사고에 관한 표현

        • 원하지 않는 사상(undesired event)
        • 비효율적인 사상(unefficient event)
        • 비계획적인 사상(unplanned event)
        • 변형된 사상(strained event)

      2. 존 듀이(John Dewey)의 5단계 사고과정

        1. 시사(suggestion)
        2. 지식화(intellectualization)
        3. 가설설정
        4. 추론(reasoning)
        5. 가설검토

      3. 인간의 동작 특성

        • 내적 조건 : 근무경력, 경험, 적성, 개성, 생리적 조건(피로, 건강)
        • 외적 조건 :

      4. 사고경향성자의 유형 ★★★

        • 상황성 누발자 : 심신의 걱정, 주의력 집중의 혼란, 작업의 난이도, 기계설비 결함
        • 습관성 누발자 : 슬럼프, 재해 경험
        • 미숙성 누발자 : 환경 미 적응 및 기능 미숙
        • 소질성 누발자 : 개인의 능력, 성격, 지능

      5. 소질적 사고 요인 : 시각기능, 지능, 성격

      • 정신상태 불량에 의한 사고 요인
        • 생리적 현상
          • 신경계통의 이상
          • 육체적 능력의 초과
          • 시력 및 청각의 이상
        • 심리적 현상
          • 과도한 자존심과 자만심

      2.2.2. 노동과 피로

      ① 피로의 증상 및 대책 ② 피로의 측정법 ③ 작업강도와 피로 ④ 생체리듬

      • 생체리듬(biorhythm) ★
        • 육체적 리듬(P, 청색 실선, 23일 주기)
        • 감성적 리듬(S, 적색 실선, 28일 주기)
        • 지성적 리듬(I, 녹색 일점쇄선, 33일 주기)

      2.2.3. 집단관리와 리더십

      ① 리더십의 유형 ② 리더십의 기법 ③ 헤드십 ④ 사기와 집단역학

      • 리더십의 유형 ★
        1. 관계지향적
          1. 우호적이며 가까이 하기 쉬움
          2. 집단구성원들을 동등하게 대함
          3. 어떤 결정에 대해 자세히 설명해줌
        2. 과업지향적
          1. 집단구성원들의 활동을 조정
        • 민주적 리더십 : 구성원과 함께 책임을 공유하고 인간에 대해 높은 관심을 가짐
      • 리더십과 헤드십 ★★
        • 리더십 : 지도자와 집단 구성원 간 유대감이 형성되기 쉬움. “같이 하자”
        • 헤드십 : 지도자와 집단 구성원 간 유대감 형성이 어려움. 공식적인 권한을 행사하는 것. “나를 따르라”
      • 리더의 권력 ★
        • 강압적 권력 : 다양한 제재와 처벌, 부정적인 결과등을 통해서 타인에게 영향력 행사
        • 보상적 권력 : 물질적 보상에 기반한 리더의 권력 행사
        • 합법적 권력 : 조직이 부여한 권한과 지위를 바탕으로 영향력 행사
        • 전문적 권력 : 전문적 기술이나 지식, 정보를 공유하여 영향력 행사
        • 준거적 권력 : 인간적인 매력을 기반으로 직원들로부터 헌신과 동일화 이끌어냄
      1. 피들러(Fiddler)의 상황 리더쉽
        • 리더의 특성 또는 행위는 주어진 상황에 따라 달라진다는 이론
        • 리더의 성격적 특성과 상황의 호의도 및 적합성의 정도에 따라 집단의 성과가 나타남
        • 상황변수 : 리더-부하의 관계, 집단의 과업구조, 리더의 직위권력
      • 허시(Hersey)와 브랜차드(Blanchard)의 상황적 리더십 이론
        • 지시형, 위임형, 참여형, 설득형
      1. 집단간 갈등의 해소방안
        • 공동의 문제 설정
        • 상위 목표의 설정
        • 집단 간 접촉 기회의 증대
        • 사회적 범주화 편향의 최소화
      2. 관리 그리드 이론(관리 격자 이론) ★
        • 바닥판 모양의 격자를 이용하여 인간과 과업 두 차원에 기초하여 리더십 이론을 설명
        • (1,1) 무관심형 : 생산과 인간에 대한 관심이 모두 낮음
        • (5,5) 중간형 : 과업과 인간의 절충
        • (1,9) 인기형 : 인간에 관한 관심은 높으나, 생산에 대한 관심은 낮음
        • (9,1) 과업형 : 생산에 관한 관심은 높으나, 인간에 대한 관심은 낮음
        • (9,9) 이상형 : 구성원에게 조직의 공동목표 및 상호의존적 관계 강조, 상호신뢰적

          • 보상 : 계획이 성취되는 데 오는 자존감
          • 합리화 : 자신의 행동에 그럴듯한 이유를 붙이는 것(변명)

          • 퇴행 : 위협이나 불안을 느끼는 상황에서 생해 초기에 만족했던 시절을 상기하는 것
          • 고립 : 타인과의 접촉을 피하고 자신만의 세계로 피하려는 것

          • 간접적

        • 패널 디스커션(panel discussion) : 토론회의 일종으로 특정한 문제사안에 대해 다른 의견을 가진 대표가 좌담회 형태로 방청자 앞에서 토의한 후 질문 등을 받으며 토론을 진행하는 것
        • 심포지엄(Symposium)
        • 버즈세션(Buzz Session)
      • 용어정리
        • 회상(recall)
        • 파지(retention) : 기억하고 있는 것들 중에서 재생되거나 또는 동일한 내용을 재학습할 경우 잠재적 효과가 나타남
        • 재인(recognition)
        • 기명(memorizing)
      • 프로그램 학습법(Skimmer) ★
        • 학습부진아의 완전학습을 위해 강화이론과 학습내용 조직의 계열성 원리 적용
        • 학습자의 능력과 진도에 따라 개별적 학습 가능한 교수기계 사용
        • 수강생의 사회성이 결여되기 쉬움
      • 테일러(Taylor) 방식(과학적관리법)

        • 인간의 기계화 및 개인차 무시를 통한 폐해 발생
      • 용어정리
        • 모방(Imitation) : 타인의 행동, 판단에 가까운 행동과 판단을 취하는 것
        • 투사(Projection) : 타인에게 책임을 전가하는 것
        • 암시(Suggestion) : 타인으로부터 판단, 행동을 무비판적으로 받아들이는 것
        • 동일화(Identification) : 타인과 자신의 비슷한 점을 발견하는 것

      <직무수행평가>

      1. 직무와 관련된 정보
        • 직무기술서 : 직무명칭, 직무의 직종, 수행되는 과업, 직무수행 방법·절차, 직무수행에 필요 도구 및 자입, 작업조건
        • 직무명세서 : 작업자에게 요구되는 능력, 기술, 적성, 지식, 성격, 흥미, 가치, 경험, 자격요건, 태도 등
      • 직무수행평가에 대한 효과적인 피드백의 원칙
        • 피드백의 효과는 항상 긍정적이지 않음
        • 개인의 수행 성과 뿐만 아니라 집단의 수행 성과에도 영향을 줌
        • 긍정적인 피드백을 먼저 제시하고, 부정적인 피드백을 제시하는 것이 효과적
        • 직무수행성과가 낮을 때 원인을 능력 부족보단 노력 부족의 탓으로 돌리는 게 효과적
      • 직무수행에서의 훈련성과
        • 교육 타당도(학습준거) : 교육 참여자들이 처음 설정한 목표를 달성하였는지에 관한 여부
        • 전이 타당도(외적준거) : 직무에 복귀한 후에 실제 직무수행에서 교육효과를 보이는 정도
        • 조직 간 타당도(외적일반화) : 교육이 현 조직 외에 타 조직에서도 효과가 있는지에 관한 여부
        • 조직 내 타당도(내적일반화) : 교육이 조직 내 타 집단에도 실시한 경우의 효과 여부
      1. 직무수행평가 시 오류
        • 후광오류 : 피평가자에 대해 잘 알지 못하는 상태에서 좋게 평가함
        • 엄격화 오류 : 피평가자에 대해 엄격히 평가함
        • 중앙집중 오류 : 피평가자들의 평가 격차가 적게 평가함
        • 관대화 오류 : 피평가자들에게 전반적으로 양호하게 평가함
      • 작업표본(work sample)
        • 실제로 작업자에게 해당 작업행위를 시켜보는 방식
        • 경력자 선발에 적합
        • 기계를 다루는 직무에 효과적
        • 시간과 비용이 많이 듦
